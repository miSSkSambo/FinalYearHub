**FinalYearHub** is an AI-inspired, interactive web platform designed to
help university students discover, explore, and select innovative final
year project ideas.

The system provides structured project recommendations across multiple
domains including Artificial Intelligence, Cybersecurity, Web
Development, Mobile Development, Data Science, IoT, and Blockchain.

intelligent filtering to create a smart academic project discovery
experience.
